安裝Homebrew軟件包版本管理器

1) 首先開啟iterm2

Terminal
$ /usr/bin/ruby -e "$(curl -fsSL
https:// raw.githubusercontent.com/ brHomebrew/install/master/install)"

2) 過程中，視窗可能會出現以下指示，要求您按Return，即 Enter

Terminal
Press RETURN to continue or any other key to abort


注意！Homebrew 沒有安裝成功，請先暫停操作回報！沒有 Homebrew 無法正確安裝 Ruby 和 Rails！
